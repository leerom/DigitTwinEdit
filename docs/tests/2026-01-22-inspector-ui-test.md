# 属性检视器 (Inspector) UI 测试文档

**文档目的**: 验证属性检视器（Inspector）在单选、多选及不同对象类型下的显示与交互行为是否符合设计要求。
**参考文档**: `docs/plans/2026-01-22-inspector-enhancement-design.md`

---

## 1. 测试环境准备

*   **前置条件**:
    *   启动应用程序。
    *   场景中至少包含以下对象：
        *   1 个相机 (Main Camera)
        *   2 个灯光 (Directional Light, Point Light)
        *   2 个模型 (Cube A, Cube B - 位置不同)

## 2. 测试用例

### TC001: 单选物体 - 变换属性 (Transform)

| 步骤 | 操作 | 预期结果 | 状态 |
| :--- | :--- | :--- | :--- |
| 1 | 在层级视图选中 "Main Camera" | 1. 检视器显示 "Main Camera" 名称。<br>2. 显示 "几何变换 (Transform)" 面板。<br>3. 旋转 (Rotation) 显示为角度值 (例如 0, 0, 0) 而非弧度。 | [ ] |
| 2 | 修改位置 X 为 5 | 1. 输入框显示 5。<br>2. 场景中相机位置更新。<br>3. 再次选中其他物体后切回，值仍为 5。 | [ ] |
| 3 | 修改旋转 Y 为 90 | 1. 输入框显示 90。<br>2. 场景中相机旋转 90 度。<br>3. 内部存储应转换为弧度 (约 1.57)。 | [ ] |

### TC002: 单选物体 - 相机属性 (Camera)

| 步骤 | 操作 | 预期结果 | 状态 |
| :--- | :--- | :--- | :--- |
| 1 | 在层级视图选中 "Main Camera" | 1. 检视器下方显示 "相机设置 (Camera)" 面板。<br>2. 显示 FOV, Near, Far 输入框。 | [ ] |
| 2 | 修改 FOV 为 90 | 1. 场景视图中相机视角变宽。<br>2. store 中 camera 组件的 fov 更新为 90。 | [ ] |
| 3 | 选中 "Directional Light" | 1. "相机设置" 面板消失。<br>2. 出现 "灯光设置 (Light)" 面板。 | [ ] |

### TC003: 单选物体 - 灯光属性 (Light)

| 步骤 | 操作 | 预期结果 | 状态 |
| :--- | :--- | :--- | :--- |
| 1 | 在层级视图选中 "Directional Light" | 1. 显示 "灯光设置 (Light)" 面板。<br>2. 显示 Color, Intensity 属性。<br>3. Range 和 Decay 应隐藏 (因方向光不支持)。 | [ ] |
| 2 | 修改 Intensity 为 2.0 | 1. 场景变亮。<br>2. 数据更新。 | [ ] |
| 3 | 勾选 Cast Shadow | 1. 阴影投射开启。 | [ ] |

### TC004: 多选物体 - 混合变换 (Transform Mixed)

| 步骤 | 操作 | 预期结果 | 状态 |
| :--- | :--- | :--- | :--- |
| 1 | 选中 "Cube A" (Pos: 0,0,0) 和 "Cube B" (Pos: 10,0,0) | 1. 检视器标题显示 "2 Objects Selected" (或类似)。<br>2. 名称输入框禁用。<br>3. 变换面板位置 X 显示 `---` (混合状态)。<br>4. 位置 Y, Z 显示 `0` (相同值)。 | [ ] |
| 2 | 在位置 Y 输入 5 | 1. 两个物体的位置 Y 都变为 5。<br>2. 场景中两个物体同时垂直移动。 | [ ] |

### TC005: 多选物体 - 类型一致性 (Type Consistency)

| 步骤 | 操作 | 预期结果 | 状态 |
| :--- | :--- | :--- | :--- |
| 1 | 选中两个灯光 (Light A, Light B) | 1. 显示 "变换 (Transform)" 面板。<br>2. **显示** "灯光设置 (Light)" 面板 (因为类型一致)。<br>3. 若 Intensity 不同，显示 `---`；若相同，显示值。 | [ ] |
| 2 | 修改 Intensity | 1. 两个灯光的强度同时更新。 | [ ] |
| 3 | 加选一个相机 (Light A + Light B + Camera) | 1. 显示 "变换 (Transform)" 面板。<br>2. **隐藏** "灯光设置" 和 "相机设置" 面板 (因为类型不一致)。 | [ ] |

### TC006: 输入框交互体验 (Input UX)

| 步骤 | 操作 | 预期结果 | 状态 |
| :--- | :--- | :--- | :--- |
| 1 | 选中任意物体，聚焦位置 X 输入框 | 1. 输入框获得焦点。 | [ ] |
| 2 | 输入 "1." (带小数点) | 1. 小数点显示正常，不会被自动删除或四舍五入。 | [ ] |
| 3 | 继续输入 "5" (即 "1.5") | 1. 显示 "1.5"。 | [ ] |
| 4 | 按下 Enter 或点击外部区域 | 1. 值提交，场景更新。 | [ ] |
| 5 | 输入非法字符 "abc" 并回车 | 1. 恢复为修改前的数值。 | [ ] |
