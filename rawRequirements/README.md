# 原始需求文档目录

本目录包含数字孪生三维场景编辑器项目的原始需求文档及UI设计参考文件。

## 目录结构

```
rawRequirements/
├── README.md                                    # 本文件
├── SceneView功能需求.md                          # Scene View详细功能需求
├── 三维场景编辑器功能需求.md                      # 整体界面布局需求
├── 场景编辑器(Scene View)操作指南.md             # 操作指南与快捷键
└── UI_Sample/                                   # UI设计参考示例
    └── code.html                                # 前端UI实现参考代码
```

## 文件说明

### 核心需求文档

#### `SceneView功能需求.md`
**类型**: 详细功能需求文档
**主要内容**:
- Scene View内部功能按钮布局设计(ASCII图示)
- 整体视图基础需求(独立窗口、鼠标键盘支持、跨系统适配)
- 右上角渲染模式工具栏功能(线框/标准/线框+标准三种模式)
- 左侧核心编辑工具栏功能(Q抓手/W移动/E旋转/R缩放/Y综合变换)
- 右下角三维坐标系辅助工具功能(轴向切换、视角复位、坐标标识)
- 核心操作功能详细说明:
  - 视图导航操作(选择、旋转、缩放、飞越漫游、聚焦跟随)
  - 游戏物体编辑操作(批量操作、属性编辑)
  - 模式适配与补充操作(2D/3D差异、跨系统快捷键)
- 交互联动与视觉反馈需求

**适用场景**: 实现Scene View场景视图的详细开发依据

---

#### `三维场景编辑器功能需求.md`
**类型**: 整体界面布局需求文档
**主要内容**:
- 顶部区域设计(标题栏 + 工具条菜单)
- 中部核心视图区设计:
  - Hierarchy(层级视图) - 左侧面板
  - Scene View(场景视图) - 中央编辑区
  - Inspector(检视器) - 右侧面板
- 底部资源管理区设计(Project/Resources双Tab页):
  - Project窗口 - 项目内资源管理
  - Resources窗口 - 公共资产库管理
- 界面布局ASCII图示及Markdown表格详细说明

**适用场景**: 整体UI框架设计与窗口布局规划

---

#### `场景编辑器(Scene View)操作指南.md`
**类型**: 用户操作指南文档
**主要内容**:
- 视图导航操作说明:
  - 基础视角控制(选择、平移、旋转、缩放)
  - 飞越漫游模式(3D模式专属)
  - 视角聚焦与定位(F聚焦、Shift+F跟随、Shift+点击复位)
- 游戏物体编辑操作说明:
  - 核心工具快捷键(Q/W/E/R/Y)
  - 物体选择与编辑快捷键(Ctrl+D复制、Delete删除、F2重命名等)
- 关键补充说明:
  - Shift加速操作
  - Mac系统快捷键适配(Ctrl→Command, Alt→Option)
  - 2D模式限制说明

**适用场景**: 开发交互功能实现参考、用户手册编写基础

### UI设计参考

#### `UI_Sample/code.html`
**类型**: HTML前端UI实现代码
**技术栈**:
- TailwindCSS (通过CDN引入,启用forms/typography/container-queries插件)
- Google Fonts: Inter字体(界面)、JetBrains Mono字体(等宽代码)
- Material Symbols Outlined图标库

**主要内容**:
- 完整的编辑器界面HTML/CSS实现
- 深色主题配色方案:
  - Primary: #3b82f6 (蓝色)
  - Accent: #f59e0b (橙色)
  - Background: #0c0e14 (深灰黑)
  - Panel: #161922 (面板深灰)
  - Border: #2d333f (边框灰)
- 界面布局实现:
  - Header顶栏(品牌TWINENGINE、菜单导航、发布按钮)
  - Hierarchy层级视图(左侧256px宽)
  - Scene View场景视图(中央弹性布局,包含工具栏、渲染模式切换、坐标系工具)
  - Inspector检视器(右侧320px宽,Transform/Materials/数字孪生数据面板)
  - Project资源窗口(底部256px高,双Tab页设计)
  - Footer状态栏(项目路径、同步状态、FPS等信息)
- 视觉元素实现:
  - 三维坐标轴Gizmo(红X/绿Y/蓝Z颜色标识)
  - 网格背景(40px方格,5%白色透明度)
  - 场景统计信息显示(Draw Calls/Tris/Verts)
  - 自定义滚动条样式(4px宽度)

**适用场景**: 前端UI开发参考、视觉设计规范、组件样式实现

## 文档阅读建议

### 首次了解项目
1. 先阅读`三维场景编辑器功能需求.md` - 了解整体界面布局
2. 再阅读`SceneView功能需求.md` - 深入理解Scene View核心功能
3. 查看`UI_Sample/code.html` - 直观感受界面视觉效果

### 开发实现阶段
1. **UI框架开发**: 参考`三维场景编辑器功能需求.md` + `UI_Sample/code.html`
2. **Scene View开发**: 参考`SceneView功能需求.md` + `场景编辑器(Scene View)操作指南.md`
3. **交互功能开发**: 参考`场景编辑器(Scene View)操作指南.md`

### 用户文档编写
直接参考`场景编辑器(Scene View)操作指南.md`,已按用户视角组织内容。
