# 基于节点的可视化材质编辑器

## 概述
提供一个可视化工具，旨在让用户无需编写任何 GLSL 或 WGSL 代码即可创建自定义着色器（Shaders）。它允许你通过在图表中连接功能块（节点）来构建复杂的视觉效果，并提供实时结果预览。

## 用户入口
+ 用户在编辑器“项目”-“Materials"中右键菜单"新建材质"弹出对话框的"新建材质类型"下拉菜单中增加"NodeMaterial"选项，以便用户创建自定义的节点材质类型。
+ 用户点击NodeMaterial类型的材质，右侧“属性检视器”界面里面只显示一个“材质编辑”按钮，用户点击后打开材质编辑器界面，以便用户对该材质进行编辑。
+ 用户双击NodeMaterial类型的材质，则直接打开材质编辑器界面。

### 材质编辑器界面各部分内容与功能解释
1. 左侧面板：节点库 (Node Library / Block List)

    这是编辑器所有“零件”的仓库。
   - 搜索框：由于节点种类繁多，可以在顶部快速输入关键字查找所需节点。
   - 分类目录：节点被分门别类地折叠在不同的手风琴菜单中，参考three.js的NodeMaterial的定义；主要的类别包括：
     - Inputs (输入)：提供基础数据类型，如浮点数(Float)、颜色(Color3/Color4)、二维/三维向量(Vector2/3/4)、时间(Time)和纹理(Texture)等。
     - Math (数学)：包含标准数学运算（加减乘除）、科学运算（三角函数、指数）和向量运算（点乘、叉乘、反射等）。
     - Matrices (矩阵)：世界矩阵、视图矩阵、投影矩阵等，用于空间坐标系转换。
     - Mesh (网格)：获取当前 3D 模型的属性，例如顶点位置(Position)、法线(Normal)、UV坐标等。
     - Output Nodes (输出)：材质的最终输出端，例如顶点输出 (VertexOutput) 和片元/像素颜色输出 (FragmentOutput)。
     - PBR：用于构建基于物理的渲染材质节点（金属度、粗糙度、清漆、各向异性等）。
    - 操作方式：用户需要将这些节点拖拽（Drag & Drop）到中间的画布区进行组合。

2. 中间区域：节点图编辑器 (Main Graph Canvas)

    这是你进行材质编程的核心工作区。
    - 节点可视化表示：被拖入画布的节点会显示为一个带有输入端口（左侧）和输出端口（右侧）的卡片。
    - 数据流连线：用户用鼠标从一个节点的输出端口拖出连线，连接到另一个节点的输入端口。数据流向遵循从左到右的原则。
    - 功能组合：通过连接，例如把“网格位置 (Mesh Position)”和“世界视图投影矩阵 (WorldViewProjectionMatrix)”相乘，最后连接到“顶点输出 (VertexOutput)”，就能决定模型在屏幕上的形状；连接颜色和光照公式到“片元输出 (FragmentOutput)”，就能决定表面的质感。

3. 右侧下方面板：实时预览区 (Preview Area)
    
    用于即时查看你当前节点逻辑渲染出的材质效果。
    - 3D 视口：一个微型的 WebGL 渲染窗口，你的材质逻辑会实时应用到中间的模型上。
    - 模型切换：你可以选择默认的预览模型，如立方体 (Cube)、圆柱体 (Cylinder)、平面 (Plane)、带有复杂曲面的材质球 (Shader ball) 或球体 (Sphere)。也可以点击 "Load..." 导入你自己的三维模型进行测试。

4. 右侧上方面板：全局属性与文件操作 (Properties & Options Panel)
    
    根据你当前在画布中选中的对象不同，此面板的内容会动态改变。如果未选中任何具体节点，它会显示整个材质图的全局设置：
    - GENERAL (通用设置)：可以修改材质名称，选择着色器类型（Standard Material标准材质、Post Process屏幕后处理、Particle粒子效果等），并可切换底层图形 API 引擎（WebGL 或更现代的 WebGPU）。
    - UI (视图控制)：
        - Zoom to fit：一键缩放画布，让所有节点都完整显示在视野内。
        - Reorganize：自动重排/对齐节点，让混乱的连线变得整洁。
        - Reset to default：清空当前画布，恢复到最初的默认状态。
    - FILE (文件操作，非常核心)：
        -Save/Load：将你的节点图保存为本地的 JSON 文件，或者读取之前保存的 JSON。
        - Generate code：极其强大的功能。它可以一键将你连好的节点图转化为能在 Babylon.js 引擎中直接运行的 JavaScript/TypeScript 代码，方便你集成到前端项目中。
        - Export shaders：直接导出底层的 GLSL 顶点着色器和片元着色器代码。

    节点参数面板 (动态显示)：如果你在中间画布选中了某个具体的节点（例如一个 Color3 节点），这个面板会被替换为该节点的属性编辑器，你可以通过颜色拾取器、滑块或输入框来改变具体数值。

## SceneView界面联动
+ SceneView中支持用NodeMaterial类型的材质进行渲染。
+ 用户对NodeMaterial类型材质的修改，将实时反应到SceneView中。
+ NodeMaterial材质编辑后，也保存到场景文件中。
