# 数字孪生/3D场景编辑工具界面说明（Markdown 表格版）
## 一、顶部区域（标题 + 工具条）
| 区域         | 内容说明                                                                 |
|------------|----------------------------------------------------------------------|
| **标题栏**   | 显示工具名称/项目名称（示例：标题）|
| **工具条**   | 「场景」「编辑」「位置」「对象变换」「配置」「帮助」（核心功能入口，点击切换对应操作面板） |

## 二、中部核心视图区（层级视图 + 场景视图 + 检视器）
| 视图名称               | 位置   | 核心功能 & 内容示例                                                                 |
|------------------------|--------|----------------------------------------------------------------------------------|
| **Hierarchy（层级视图）** | 左侧   | 展示场景内所有对象的层级结构，支持选中、展开/折叠、父子级管理<br>示例：<br>- SampleScene（场景名称）<br>　├─Main Camera（主相机）<br>　├─Directional Light（平行光）<br>　└─Cube (Selected)（选中的立方体） |
| **Scene View（场景视图）** | 中间   | 3D场景可视化编辑区，显示物体、光源、相机等，支持旋转/平移/缩放操作<br>核心元素：<br>- 三维场景编辑区<br>- 选中对象的 Transform Gizmo（变换控件）<br>- 场景内物体与光源实时预览 |
| **Inspector（检视器）**   | 右侧   | 展示选中对象的属性面板，支持参数修改<br>示例：<br>- Transform（位置/旋转/缩放）<br>　├─......（具体参数）<br>　└─......<br>- Material（材质）<br>　├─......<br>　└─...... |

## 三、底部资源管理区（Project / Resources 双 Tab 页）
| Tab 页名称       | 子区域               | 核心定位 & 功能说明                                                                 | 内容示例                                                                 |
|------------------|----------------------|----------------------------------------------------------------------------------|----------------------------------------------------------------------|
| **Project**      | Project Window（项目窗口） | **项目内资源管理**：展示当前项目已引用/使用的资源，用于场景内对象的编辑、替换、配置 | - Assets（根目录）<br>　├─Scenes(Selected)（项目场景）<br>　├─Materials（项目材质）<br>　└─TwinBody（项目内孪生体） |
|                  | Content Window（内容窗口） | 显示当前选中目录下的具体资源文件，支持拖拽到场景/层级视图使用                         | - Scene1.json<br>- Scene2.json<br>- 项目内材质/模型文件等             |
| **Resources**    | Resources Window（资源库窗口） | **公共资产库管理**：展示数字孪生资产库中的公共资源，用于项目内资源的引用、导入、复用     | - 公共模型库（建筑/设备/部件）<br>- 公共材质库（PBR/标准材质）<br>- 公共场景模板 |
|                  | Content Window（内容窗口） | 显示资产库中对应分类下的资源，支持预览、筛选、拖拽到 Project 或 Scene 中使用             | - TwinModel_A.glb<br>- TwinMaterial_B.mat<br>- PublicScene_Template.json |

---
以上为数字孪生/3D场景编辑工具的界面布局说明，涵盖顶部工具条、中部核心视图区及底部资源管理区的功能与内容示例，帮助用户快速理解和使用该编辑器进行三维场景的创建与管理。
Scene View（场景视图）内的具体功能参考相关文档说明。
