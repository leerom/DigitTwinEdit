===========================================
🐳 Docker Desktop 快速安装指南
===========================================

Docker Desktop 未安装。请按以下步骤操作：

┌─────────────────────────────────────────┐
│  方式1: 自动下载安装 (推荐)              │
└─────────────────────────────────────────┘

运行:
  install-docker.bat

脚本会自动下载并启动安装程序。


┌─────────────────────────────────────────┐
│  方式2: 手动下载安装                     │
└─────────────────────────────────────────┘

1. 访问: https://www.docker.com/products/docker-desktop/

2. 点击 "Download for Windows"

3. 运行下载的安装程序

4. 安装选项:
   ✓ Use WSL 2 (推荐)
   ✓ Add shortcut to desktop

5. 完成安装并重启电脑


┌─────────────────────────────────────────┐
│  安装后操作                              │
└─────────────────────────────────────────┘

1. 重启电脑后，Docker Desktop会自动启动

2. 等待托盘图标变为绿色 (约1-2分钟)

3. 验证安装:
   docker --version
   docker ps

4. 运行PostgreSQL容器:
   start-postgres-docker.bat

5. 启动真实后端:
   cd packages\server
   pnpm dev

6. 测试完整功能!


┌─────────────────────────────────────────┐
│  预计时间                                │
└─────────────────────────────────────────┘

下载: 5-15分钟 (取决于网速)
安装: 5-10分钟
重启: 2分钟
Docker启动: 1-2分钟
PostgreSQL容器: 1分钟
────────────────────────────────
总计: 约15-30分钟
